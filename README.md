# Dataset & Data Types – Titanic Dataset

## Task Overview
This project is part of the AI & ML Internship – Task 1.
The objective of this task is to understand a dataset,
identify different data types, and analyze data quality issues
before applying machine learning techniques.

## Dataset Used
Titanic Dataset (Public Dataset)

## Tools Used
- Python
- Pandas
- NumPy

## Task Performed
- Loaded the Titanic dataset
- Identified numerical, categorical, ordinal, and binary features
- Identified the target variable
- Checked missing values and data imbalance
- Analyzed dataset suitability for machine learning

## Conclusion
This task helped in understanding the structure and characteristics
of the dataset, which is essential before building any ML model.
